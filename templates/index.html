Jenkins Redirector

Just a little service that will use the Jenkins API to discover the latest build ID for a multibranch pipeline build, and redirect to the view of said build in Blue Ocean.

It requires that you give it the job name and branch (defaults to "master"). For example:

/bedrock_multibranch_pipeline/prod/

bedrock_multibranch_pipeline is the job name, and prod is the branch.

See the Github Repo for more details and to get the code. This instance of the app is setup to work with {{ jenkins_url }}.
